# Vim configuration

## Dependencies
- Plug [https://github.com/junegunn/vim-plug](https://github.com/junegunn/vim-plug)

## How to use
- Install Plug
- Link/copy vimrc to $HOME_PATH/.vimrc
    `ln -s ./.vimrc ~/.vimrc`
    or
    `cp ./.vimrc ~/.vimrc`
- Open ~/.vimrc and run `PlugInstall` in ex mode
